fastapi-tenancy
📦 PyPI Published
⌥ GitHub ↗ PyPI

Enterprise-grade multi-tenancy library for FastAPI. Supports 4 isolation strategies — schema-per-tenant, database-per-tenant, Row-Level Security, and hybrid mode — with async-first architecture using AsyncSession and contextvars. Zero blocking I/O, auto dialect detection for PostgreSQL, MySQL, SQLite, and MSSQL.
